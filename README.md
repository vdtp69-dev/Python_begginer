# Python Beginner Projects

A collection of beginner-friendly Python projects built while learning Python fundamentals, problem-solving, and programming concepts.

## Projects Included

### 1. Contact Book

A command-line contact management application.

#### Features

* Add contacts
* Search contacts
* View all contacts
* Update contact information
* Delete contacts
* Count total contacts
* Save contacts to a file
* Load contacts automatically when the program starts

#### Concepts Used

* Dictionaries
* Functions
* File Handling
* CRUD Operations

---

### 2. To-Do List

A simple task management application.

#### Features

* Add tasks
* View tasks
* Delete specific tasks
* Delete all tasks
* Save tasks to a file
* Load tasks automatically

#### Concepts Used

* Lists
* Functions
* File Handling
* Loops

---

### 3. Password Generator

A random password generator.

#### Features

* Custom password length
* Generate multiple passwords
* Optional special characters
* Random password generation

#### Concepts Used

* Functions
* Strings
* Loops
* random module

---

### 4. Expense Tracker

A command-line expense tracking application.

#### Features

* Add expenses
* View expenses
* Show category-wise spending
* Show total spending
* Find highest single expense
* Find highest spending category
* Input validation using try/except
* Case-insensitive category handling

#### Concepts Used

* Nested Lists
* Dictionaries
* Functions
* Exception Handling
* enumerate()
* Data Aggregation
